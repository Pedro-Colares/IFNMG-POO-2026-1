# 📚 Atividades de Programação Orientada a Objetos (POO)

Repositório criado para armazenar atividades e projetos desenvolvidos durante a disciplina de **Programação Orientada a Objetos (POO)** em Java.

## 🎯 Objetivo

Este repositório tem como objetivo:
- Praticar os conceitos fundamentais de POO
- Organizar os códigos desenvolvidos ao longo da disciplina
- Servir como material de estudo e revisão
- Acompanhar minha evolução na linguagem Java

## 📂 Projetos

Cada pasta deste repositório representa um projeto independente.  
Dentro de cada uma, há um README com explicações detalhadas sobre a implementação.
